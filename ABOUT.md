# WaterDepthGauge

**Version:** 1.0.2
**Author:** BeyondSmash
**GitHub:** https://github.com/BeyondSmash/WaterDepthGauge

## What is this?

An underwater depth meter HUD for Hytale that shows:
- Local water depth (from entry point)
- Sea level depth (from Y=115)
- Direction indicators (rising/descending)

## Installation

1. Download `WaterDepthGauge-1.0.2.jar` from [Releases](https://github.com/BeyondSmash/WaterDepthGauge/releases)
2. Place in `%AppData%\Hytale\UserData\Mods\`
3. Restart server

## Commands

- `/wdepth on` - Enable HUD
- `/wdepth off` - Disable HUD
- `/wdepth status` - Check status

## Compatible Mods

Pre-configured for:
- BlockInfo
- AdvancedItemInfo
- AdminUI
- EyeSpyHUD

For other mods, see [README.md](README.md) for fix instructions.
